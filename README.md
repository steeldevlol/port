# Steeldev's Port

**Port is used to create tunnels like serveo if you dont have port forwarding**

# Installing Port

Just run this command: **apt update && apt install git -y && cd ~ && git clone https://github.com/steeldevlol/port && chmod +x port/port && mv port/port /bin/port && clear && rm -rf ~/port && echo 'âœ… port installed successfully, type "port help" to use it.'**
and port should be installed!

# How to use Port?

Run _port help_ to see all commands for port.
